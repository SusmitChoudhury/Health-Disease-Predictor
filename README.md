# Health-Disease-Predictor (Python CLI Project)

A simple, interactive Python-based health analysis program that checks a person’s basic health metrics and gives meaningful suggestions based on medical ranges.
This program evaluates:
Heartbeat (bpm)
Blood Pressure (systolic mmHg)
Sugar level (mg/dL)
Body temperature (°C)
Oxygen level (%)
Based on the user’s inputs, it gives instant feedback whether each parameter is:
LOW
NORMAL
BORDERLINE
HIGH
RISKY

Features

✔ Interactive command-line health checker
✔ Shows health status for 5 vital readings
✔ Medical-based thresholds
✔ Gives improvement tips
✔ Beginner-friendly Python logic
✔ Uses conditions & comparisons
